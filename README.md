## General info
Plov is next-gen Blockchain.

## Wallet
### Installation
Use `npm` package manager to install
```bash
npm install --global plov-wallet
```
### Usage
Usage example:
```bash
$ plov status
Last time: 1622449597403
Last block: 443016
```
Use `--help` option to get more information
```bash
plov --help
```

## Node
### Installation
Use `npm` package manager to install
```bash
npm install --global plov-node
```
### Usage
Usage example:
```bash
$ plov-node sync
Sync done
```
Use `--help` option to get more information
```bash
plov-node --help
```
